# Självbevattningssystem
Python PyGame simulation för självbevattningssystem.


## Installation ##
### Windows ###
- Se till att du har Python installerat. Om inte, installera här: https://www.python.org/downloads/windows/ 
- Se till att du har pip. Om inte, installera här: https://pip.pypa.io/en/stable/installation/

#### Installera PyGame: ####
```
pip install pygame-ce
```

### Linux ###
- Se till att du har Python installerat:
```
sudo su # then enter password to make sure you are the root user
apt update # update Linux packages
apt upgrade # upgrade Linux packages
sudo apt install python3 # installerar Python
```
- Se till att du har pip installerat:
```
sudo su # then enter password to make sure you are the root user
apt update # update Linux packages
apt upgrade # upgrade Linux packages
sudo apt install python3-pip # installerar pip
```
#### Installera PyGame: ####
```
sudo su # then enter password to make sure you are the root user
apt update # update Linux packages
apt upgrade # upgrade Linux packages
sudo apt-get install python3-pygame # press Y when asked "Do you want to continue?[Y/n]"
```

### MacOS ###
- Se till att du har Python installerat. Om inte, installera här: https://www.python.org/downloads/macos/
- Se till att du har pip. Om inte, installera här: https://pip.pypa.io/en/stable/installation/

#### Installera PyGame: ####
```
pip3 install pygame
```
