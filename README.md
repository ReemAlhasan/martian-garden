# Självbevattningssystem
Python PyGame simulation för självbevattningssystem. Välj din planet!:)


### Installation ###
## Windows ##
```
pip install pygame-ce
```

## Linux ##
```
sudo su # then enter password to make sure you are the root user
apt update # update Linux packages
sudo apt-get install python3-pygame # press Y when asked "Do you want to continue?[Y/n]"
```
